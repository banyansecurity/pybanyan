# GCP Cloud Functions

Discover your GCP resources. See docs for more info.

In GCP Cloud Functions, run container every hour.


Specify:
```
BANYAN_API_URL
BANYAN_REFRESH_TOKEN
GOOGLE_CLOUD_FUNCTIONS=TRUE
```

Optionally:
```
RESOURCE_TYPE
PROJECT
ZONE
TAG_NAME
```

---

Build it:
```
docker build
```

Push it to Registry
```
docker push
```
